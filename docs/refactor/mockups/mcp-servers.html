How does checking a box change my files?
Exclusive (Cursor, Droid)

The whole MCP file is owned by Plexus. Toggling rewrites a cached copy in ~/.config/plexus/.cache/mcp/ and the agent's path symlinks to it.

Shared (Claude, Codex)

The agent file holds auth + history. Plexus only rewrites the mcpServers key in place; other keys are preserved byte-for-byte.

Every toggle takes a snapshot first. Restore from Backups.
